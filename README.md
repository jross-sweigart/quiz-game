# Quiz Game in Python

## Overview
Welcome to the Quiz Game project! This application is built in Python and allows users to take a quiz by reading questions from a CSV file. The program keeps track of the user's score, providing immediate feedback on their answers.

## Features
- **CSV-based Quiz**: Load quiz questions and answers from a customizable CSV file.
- **Score Tracking**: Count and display the number of correct and incorrect answers.
- **Timer Functionality**: Set a time limit for the quiz, with a default of 30 seconds.
- **Input Validation**: Trim whitespace and handle capitalization to ensure accurate answer checking.
- **Shuffle Questions**: Randomize the order of questions for each quiz session.
